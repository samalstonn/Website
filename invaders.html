Space Invaders Project

Finished Game

This project was for a final assignment in my class. I coded a modern version of the class game "Space Invaders" from scratch with a partner. The game was created with Python, Atom, and API from a 2D game module. Coding this game really taught me a lot as it was my first major and hands-off computer science project. It really ingrained object-oriented programming skills and code organization skills. I also got to work with a partner and see first hand how it is to work on a software project in a team.
